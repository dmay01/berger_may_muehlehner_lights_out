berger_may_muehlehner_lights_out
================================
Die Aufgabenstellung ist, das Spiel "Lights out" zu programmieren.
Das Spiel besteht aus einem 5x5 Gitter. Anfangs sind zufällig viele "Lichter" eingeschalten. Mit einem Klick auf ein Licht werden das betroffene Licht und alle umliegenden Lichter umgeschaltet.
Diagonale Lichter sind davon nicht betroffen. Das Ziel ist es mit so wenig Klicks wie möglich alle Lichter auszuschalten.

Das Team besteht aus Berger, May und Mühlehner.
Die Aufgaben sind wie folgt verteilt:
Berger: View, Protokoll
May: Model, Protokoll
Mühlehner: Controller, Protokoll
